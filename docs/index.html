Opuza

Opuza is a project that allows anyone to sell files on the web for bitcoin using the Lightning Network. Everything on this site is served by an Opuza instance run by the Opuza developers. Opuza is free and open source, so anyone can run their own Opuza instance.

Opuza's code is hosted on GitHub.

Feedback is greatly appreciated. If you run into a bug or have a feature request, feel free to open an issue.

We publish periodic updates to our blog.
